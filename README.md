## 💾 データ管理

### LocalStorage キー一覧
各ゲームは以下のキーでランキングデータを保存しています:

| ゲーム | ストレージキー | データ形式 | ソート順 |
|-------|---------------|-----------|---------|
| もぐら叩き | `mogura_ranking_v2` | 数値（点数） | 降順（大きい順） |
| 絵合わせ | `memory_ranking_v1` | 数値（手数） | 昇順（小さい順） |
| 数字タッチ | `numbers_ranking_v1` | 数値（秒数） | 昇順（小さい順） |
| 計算ドリル | `calc_ranking_v1` | 数値（正解数） | 降順（大きい順） |
| 色並べ | `puzzle_ranking_v1` | 数値（秒数） | 昇順（小さい順） |
| 光の順番 | `light_memory_ranking_v1` | 数値（レベル） | 降順（大きい順） |
| 仲間はずれ | `odd_one_ranking_v1` | 数値（個数） | 降順（大きい順） |

### Googleスプレッドシート連携
`common.js`の`sendData()`関数を使用して、各ゲームの結果を外部スプレッドシートに送信しています。

```javascript
sendData("ゲーム名", スコア);
```
